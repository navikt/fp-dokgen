# Du må søke om å endre svangerskapspengeperioden din
<p style="color:gray">
Saksnummer i NAV: 123456789
<br/>
Navn: Dolly Duck
<br/>
Fødselsnummer: 300220 12345
<br/>

</p>

Vi har fått opplysninger om at du jobber samtidig som du får svangerskapspenger. Derfor vurderer vi å stanse svangerskapspengene dine.

Du kan ikke jobbe heltid og samtidig få svangerskapspenger. Arbeider du deltid, kan du søke om å kombinere svangerskapspengene med arbeid.

Ønsker du å søke om å endre svangerskapsperioden eller å uttale deg, må du gjøre dette innen 24. mai 2020.

Du kan søke om å endre perioden din på [nav.no](https://nav.no/). Ønsker du å uttale deg kan du gjøre det ved å logge deg inn og velge «Skriv til oss» på [nav.no/skriv-til-oss](https://www.nav.no/skriv-til-oss).

Opplysningsplikten går fram av folketrygdloven §§ 21-3 og 21-4.
<br/>
##### Vedtaket ditt om svangerskapspenger kan bli omgjort
Vi vil vurdere saken din på nytt når fristen er gått ut. Dersom du ikke har hatt rett til svangerskapspenger, kan du bli nødt til å betale tilbake pengene.

Du sender dokumentasjonen til oss på [nav.no/ettersendelser](https://nav.no/ettersendelser).
<br/>
<br/>
##### Du har rett til innsyn
For å se dokumentene i saken din, logg inn på [foreldrepenger.nav.no](https://foreldrepenger.nav.no/).
